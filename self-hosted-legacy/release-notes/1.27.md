## Release Notes

### 1.27.4 (2026-07-13)

This release deals with dependency versions triggering alerts in vulnerability scan tools. Please note that previous Gatling Enterprise versions actually vulnerable.

* upgrade [Netty 4.2.16.Final](https://github.com/netty/netty/releases)

### 1.27.3 (2026-06-29)

This release deals with dependency versions triggering alerts in vulnerability scan tools. Please note that previous Gatling Enterprise versions actually vulnerable.

* upgrade [Netty 4.2.15.Final](https://github.com/netty/netty/releases)
* migrate from Akka 2.6.0 to Apache Pekko 1.6.0
* remove dependencies to `io.airlift:aircompressor` and `org.codehaus.plexus:plexus-utils` (that were only used when fetching test packages from maven repositories)

Please note that Akka is still used in load generators when running Gatling versions ranging from 3.5 to 3.11 included. This cannot change for compatibility reasons. The Akka dependency was removed in Gatling 3.12.
